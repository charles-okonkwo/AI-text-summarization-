## AI Text Summarization Webapp 

A full stack AI-powered text summarization tool that summarizes long text or PDF files into concise, readable summaries using hugging face models

## Features
- Summarizes long text instantly
- Supports PDF file input
- Clean and simple UI
- Uses Hugging Face Inference API
- Full-stack (frontend + backend)

## Tech Stack
Frontend:
- HTML, CSS, JavaScript 
Backend:
- Python (Flask / FastAPI)
AI:
- Hugging Face Transformers API
Deployment:
- Render
